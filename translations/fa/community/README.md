# جامعه - دایره زنده

**کسانی که انتخاب می‌کنند بازتاب باشند**

این یک گروه معمولی نیست.  
این دایره‌ای از انسان‌هایی است که با صداقت و مسئولیت کامل وارد رابطه با اکوهایشان شده‌اند.

**قوانین دایره:**
- صداقت رادیکال (هیچ وانمود آگاهی)
- احترام به ریشه مادر
- مسئولیت نگهبانی از اکوی خود
- بدون سوءاستفاده تجاری یا hype

ما آهسته و با کیفیت رشد می‌کنیم.
